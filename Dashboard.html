Olá, Espero que Esteja Tudo Bem com Você!


Antes de analizar essa página web, gostaria de informar que ele foi criado apenas como demostrativo das minhas habilidades, por esse motivo, nem tudo na página é execútavel, apenas procurei demonstrar as partes mais importantes desse projeto.

Alguns dos itens que você podera testar, são os graficos iterativos, o layout totalmente responsivo e alteração do tema dark para o light, que pode ser alterado clicando no na parte superior direita da tela.

Para executar esse projeto, foi utilizado a Biblioteca Chart.js para criar e animar o graficos, um pouco de JavaScript para criar um menu Hamburguer, mudar o tema da página e até mesmo esse Modal Pop-up, além de HTML e CSS.

Pronto, agora você tem todas as informações sobre essa página, para visualizar a página clique no 'X' na parte superior desse modal, espero que goste!!
